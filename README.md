# HabitForge 🧠

A simple habit tracking CLI application built with Python.

## Features
- Create and track habits
- Store data using SQLite
- View completion statistics
- Identify your most productive days

## Technologies
- Python
- SQLite

## How to Run
```bash
python main.py
```

## Project Structure
```
habit_tracker/
│
├── main.py
├── database.py
├── habits.py
├── analytics.py
├── utils.py
└── data.db
```
